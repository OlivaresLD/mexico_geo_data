# México GeoData
Repositorio con links a las fuentes de datos geográficas de México.

### Modelos digitales de elevación:
* [Continuo Elevaciones Mexicano (INEGI)](https://www.inegi.org.mx/app/geo2/elevacionesmex/)

### Marco Geodésico
* [Geoide Gravimétrico Mexicano (INEGI)](https://www.inegi.org.mx/temas/geoide/)
* [Procesamiento GNSS(INEGI)](https://www.inegi.org.mx/temas/gnss/)
* [Red Geodésica Nacional Activa (INEGI)](https://www.inegi.org.mx/temas/geodesia_activa/)

### Marco Geoestadístico
* [Marco Geoestadístico](https://www.inegi.org.mx/temas/mg/)

### Recursos Naturales
* [Áreas Naturales Protegidas](http://sig.conanp.gob.mx/website/pagsig/info_shape.htm)
* [GeoInfoMex 2D](https://www.sgm.gob.mx/GeoInfoMexGobMx/)
* [GeoInfoMex 3D](https://www.sgm.gob.mx/GeoInfoMex3D/)
